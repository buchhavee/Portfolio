TEMA 02 // WEB

WEB

Gå til projekt

Dette tema var en introduktion til arbejde med website efter mobile first princippet. Siden skulle udarbejdes efter udleverede wireframe og layoutdiagrammer. Indholdet på sitet var også udleveret, hvor 1 af siderne skulle indholde eget billede og tekst ang. ens egen computer. TEMA 03 - UX/UI

UX/UI

Gå til projekt

Dette tema omfattede idéudvikling, research, prototype/design, kodning og dokumentation. Research brugte user stories til at forstå målgruppe og formål. Moodboard, tekstindhold og billeder blev udarbejdet som grundlag for en klikbar prototype i Figma med styletile. Prototypen blev testet og kodet som en MVP i HTML, CSS og JS med forbedringer, herunder en scrollbar-sektion for butikker. Forløbet blev afsluttet med præsentation og procesdokumentation.

Tema 04 // ANIMATION

ANIMATION

Gå til projekt

Dette tema fokuserede på JavaScript og CSS-animationer med målet at udvikle et spil. Idéudviklingen omfattede brainstorm, skitsering, papirsprototype og moodboard.
Designfasen inkluderede udvikling af spillets baggrund og elementer i Illustrator, eksport i SVG-format, og design i et responsivt 16:9-format (1920x1080)
Programmeringsfasen anvendte HTML, CSS og JS til spillets mekanik og regler, understøttet af aktivitets- og State Machine-diagrammer. Spillet indeholder klikbare elementer, fastlagte baner, feedback som point og spilafslutning, samt lyd-effekter knyttet til mekanikker. Udviklingen afsluttedes med en valideret hjemmeside, der indeholder spil, grafik, assets og tekniske detaljer.

TEMA 05 // INDHOLD

INDHOLD

Passionssite
Virksomhedsite

Dette tema omfattede grundlæggende indholdsproduktion, herunder præproduktion, produktion og postproduktion. Der er arbejdet med videooptagelse, ekstern lyd og redigering i Premiere Pro. Der er blevet udarbejdet content for "kunder" i form af et passionssite og et virksomhedsite. Til virksomhedsite er der også blevet lavet et komplet redesign af eksisterende hjemmeside, som tager udgangspunkt i brand DNA.
Temaet omfatter også arbejde med animeret vektorgrafik (Lottiefiles)
